# Level 08 - File Size Filtering

## Objective
Identificar un archivo por tamano.

## Skills learned
- `find -size`

## Description
Entre varios archivos, solo uno coincide con el tamano exacto que contiene la flag.

## Hidden flag
Generada por `setup.sh` y usada como password de `dw9`.

## Optional hints
- Usa el sufijo `c` para bytes.
