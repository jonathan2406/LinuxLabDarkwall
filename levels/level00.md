# Level 00 - Basic Navigation

## Objective
Encontrar la flag navegando por el sistema de archivos del usuario `dw0`.

## Skills learned
- `pwd`
- `ls`
- `cd`
- `cat`

## Description
Debes ubicar un directorio inicial y leer un archivo oculto con la flag. Esta flag es la password de `dw1`.

## Hidden flag
Se genera automaticamente en `setup.sh` (consultable por administradores en `/opt/darkwall_lab/meta/flags.csv`).

## Optional hints
- Recorre `~/level00/start_here`.
- Usa `ls -la` si no ves todos los archivos.
